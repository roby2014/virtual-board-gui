## Virtual Board GUI

[Simulation project](https://github.com/roby2014/virtual-board-vhdl)

The GUI depends on a `board.json` file (check `src/main/resources` for an example).
Supported components:
- switches
- leds
- buttons
- otherPins
- segsev
